    AgentMesh¶

    A2A Protocol Adapter & CLI Tool — 为智能体通信构建的基础设施

    AgentMesh 是一个开源的 A2A (Agent-to-Agent) 协议适配器与命令行工具集，提供 Schema 验证、消息格式转换、保真度追踪和多 Agent 工作流编排能力。

    • Schema 验证 — 支持 Task、Message、AgentCard、Request、Response 五种 A2A Schema 格式校验
    • 双向转换 — AgentMesh 内部格式与 A2A 标准 Task/JSON-RPC 格式之间无缝转换，自动方向检测
    • 保真度追踪 — 五维度信息保留率评分（状态、文本相似度、数据字段、Artifact、元数据），可配置阈值
    • 端到端执行 — 从 JSON 配置文件定义和执行多步骤 Agent 工作流，支持 Dry-Run 预览

    快速上手¶

    # 安装
pip install agentmesh

# 初始化项目
agentmesh init my-project

# 验证 A2A Task 消息
agentmesh validate task.json --schema task -v

# 格式转换
agentmesh convert msg.json --direction to-a2a -p -o result.json

# 保真度追踪
agentmesh fidelity -i input.json -o output.json -v

# 运行工作流
agentmesh run workflow.json --dry-run

    核心架构¶

    AgentMesh 位于你的 Agent 与 A2A 协议之间，提供验证、转换、追踪和编排能力：

    • A2A Adapter — A2A 协议正向/反向转换
    • CLI (Click) — 五个核心命令
    • A2A Models — 协议数据模型
    • Agent SDK — 核心协作能力
